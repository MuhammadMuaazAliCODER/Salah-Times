🕌 Salah Times — Islamic Prayer & Calendar Guide

A beautifully designed Islamic web application that provides accurate prayer times, Sehri & Iftar countdown, Hijri date, and a monthly prayer calendar based on the user’s location.

Built using pure HTML, CSS, and JavaScript — all inside a single HTML file.

🌟 Features

📍 Automatic location detection (Geolocation API)

🌍 Detects city name using reverse geocoding

🕋 Displays all 5 daily prayers (Fajr, Dhuhr, Asr, Maghrib, Isha)

🌙 Sehri & Iftar live countdown

⏳ “Next Prayer” real-time countdown timer

📅 Monthly prayer time calendar

🗓 Hijri date (Arabic + English month names)

🌌 Animated Islamic-themed UI (stars background + moon SVG)

📱 Fully responsive design (Mobile, Tablet, Desktop)

🛠 Built With

HTML5

CSS3 (Custom Design System + Animations)

Vanilla JavaScript

Browser Geolocation API

📡 APIs Used

Aladhan Prayer Times API – For monthly prayer times

OpenStreetMap Nominatim API – For reverse geolocation

🚀 How It Works

User allows location access.

Browser provides latitude & longitude.

App fetches:

City name

Monthly prayer timings

Displays:

Today’s prayer times

Sehri & Iftar timing

Live countdowns

Full monthly calendar

Updates time and countdown every second.

If location permission is denied, the app automatically falls back to:

Karachi, Pakistan (Default Location)
